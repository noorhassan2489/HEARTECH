# Ling Six Audio Files

Place the following 6 audio files in this directory:

- `ling_m.mp3` — /m/ sound (250-500 Hz, nasal)
- `ling_ah.mp3` — /ah/ sound (500-1000 Hz, open vowel)
- `ling_oo.mp3` — /oo/ sound (500-1000 Hz, rounded vowel)
- `ling_ee.mp3` — /ee/ sound (1000-3000 Hz, front vowel)
- `ling_sh.mp3` — /sh/ sound (2000-4000 Hz, fricative)
- `ling_s.mp3` — /s/ sound (4000-8000 Hz, sibilant)
